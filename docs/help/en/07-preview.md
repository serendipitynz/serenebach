# Preview mode

Preview mode lets you check unpublished entries and try out templates that aren't activated yet. It mirrors SB3's "dynamic preview" idea.

## Previewing draft entries

The **Preview** link in the entry editor opens a draft or closed entry in the same layout as a published one.

The entry stays hidden from the public site, so you can fix layout issues or proof-read long articles without exposing them.

## Previewing templates

From the templates list or the template editor, you can render the public site with a specific template applied.

This works without making the template "active", so you can refine an alternative without disturbing the live design.

## Combining entries and templates

A draft entry can also be previewed against a not-yet-active template — useful when you're tuning an imported SB3 template or trying out a category-specific design.

## Who can use it

Preview is available only to logged-in admin users. Following a preview URL while signed out won't reveal drafts or unpublished templates.

Preview pages are flagged so search engines won't index them, and they aren't included in static rebuild output.
