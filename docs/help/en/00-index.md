# Serene Bach — Help

Serene Bach is a self-hostable weblog management tool. It carries forward the "set up on your own server, manage it yourself" approach of the 2000s-era Serene Bach / SB3, reimplemented as a single Go binary.

This help focuses on the day-to-day admin operations — writing entries, dialling in the look, managing comments, migrating from an existing SB3 site — rather than internal mechanics.

## About this help

Switch pages from the "Help" menu in the admin sidebar. The help is bundled into the Serene Bach binary, so it works offline.

The admin UI and templates broadly resemble SB3, but comments, trackbacks, image handling, and AI integrations differ in places. Before migrating, please review [Migrating from SB2 / SB3 and feature differences](sb3-migration).
