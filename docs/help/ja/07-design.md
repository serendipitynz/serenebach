# ブログデザインと OG カード

「デザイン」画面では、利用するテンプレートの切り替えや、サイト共通の表示設定、OG カードの既定値、カスタムタグ、テンプレートのインポート/エクスポートを扱います。

テンプレート 1 セット自体の編集方法は [テンプレート編集](templates) を参照してください。本ページは「どのテンプレートを使うか」「サイト全体に共通する表示の取り決め」「SNS 共有時に出る OG カード」に関する設定をまとめています。

## デザイン設定

「デザイン」では、公開サイト全体に関わる表示設定を変更できます。

- 利用中テンプレートの切り替え
- アーカイブ / カテゴリー / タグ一覧で使うテンプレート
- プロフィールページで使うテンプレート
- 一覧に表示する記事数
- 記事の並び順
- コメントの並び順
- 日付の表示形式

日付の表示形式は SB3 と同じ `%Year%` / `%Mon%` / `%Day%` / `%Week%` / `%Hour%` / `%Min%` などの書き方に対応しています。例: `%Year%-%Mon%-%Day% (%Week%)`

カテゴリーごとに別のテンプレートを使いたい場合は、各カテゴリーの編集画面でテンプレートを指定します。

## OG カード

OG カードは、SNS などで記事 URL が共有されたときに使われる画像です。Serene Bach では、記事ごとに 1200 x 630 の画像を生成します。

ブログ共通の既定値（背景画像と文字色）は、「デザイン」>「OG カード」タブで編集します。記事ごとの上書きは記事編集画面で指定できます。

背景画像は次の順で使われます。

1. 記事ごとの OG カード背景（記事編集画面で指定）
2. ブログ共通の OG カード背景（デザイン > OG カード）
3. 同梱の標準背景

背景画像は画像ライブラリから選びます。比率が合わない画像は中央を基準に切り抜かれます。

文字色は記事タイトルとブログタイトルの両方に適用されます。未設定の場合は標準の 2 色が使われます。背景画像の上に既に文字が入っているなど、OG カード側で文字を出したくない場合は「文字を表示しない」を選んでください。

## カスタムタグ

「カスタムタグ」タブでは、テンプレート内で使う独自の `{custom_xxx}` タグを登録できます。値は HTML やテキストをそのまま記述でき、テンプレートに埋め込まれるときにエスケープされません。

```html
<!-- テンプレート内でこう書く -->
<div class="analytics">{custom_google_analytics}</div>
```

管理画面で以下のように登録します。

| 項目 | ルール |
|---|---|
| タグ名 | `custom_` に続けて英小文字・数字・アンダースコアのみ（最大 50 文字） |
| 値 | HTML またはテキスト（最大 64 KB） |
| 上限 | ウェブログあたり最大 50 件 |

登録したタグは、個別記事・一覧・カテゴリー・アーカイブ・プロフィールなど**すべてのページ種別**に自動的に展開されます。値の変更は即座に公開サイトに反映されます（静的書き出しを使っている場合は再構築が必要です）。

> **セキュリティに関する注意:** 値はオペレーターが入力する HTML をそのまま出力します。公開サイトの読者が値を変更することはできないため、XSS の影響範囲は管理者権限を持つユーザーに限定されます。

## テンプレートのインポートとエクスポート

SB3 形式の `template.txt` をインポートできます。古いテンプレートで使われている Shift_JIS、EUC-JP、ISO-2022-JP などの文字コードは、取り込み時に UTF-8 へ変換されます。

エクスポートも `template.txt` 形式で行います。バックアップや別環境への移動に利用してください。
