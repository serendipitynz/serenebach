# リンク

「リンク」画面では、サイドバーや一覧で表示するリンク集（ブログロール）を管理できます。SB3 のリンク機能を引き継いだもので、テンプレートの `link` ブロックと `{link_list}` タグから利用します。

リンク機能は管理ユーザーまたは上級ユーザーが利用できます。一般ユーザーには表示されません。

## リンクとグループ

リンクには 2 種類あります。

| 種類 | 用途 |
|---|---|
| リンク | 1 つのリンク先（タイトル + URL + 開き方）を表します |
| グループ | 複数のリンクをまとめる見出しです |

グループの中にリンクを入れ子にできますが、グループの中にグループは置けません（1 階層のみ）。

## 新規追加と編集

「新規リンク」を押すと種類選択付きの作成フォームが開きます。グループ編集画面から「新規リンク」を押した場合は、そのグループのメンバーとしてリンクが作られます。

リンク 1 件の主な項目:

| 項目 | 内容 |
|---|---|
| 名前 | リンクとして表示されるテキスト |
| URL | リンク先の URL |
| 説明 | 補足説明（テンプレートが対応していれば表示されます） |
| ターゲット | `_self` / `_blank` などの `target` 属性 |
| 所属グループ | グループに入れる場合のみ指定します |
| 表示 / 非表示 | 一時的に隠したいときに使います |

グループの主な項目は名前と説明だけです。URL やターゲットはグループには付きません。

## 並び替え

リンクとグループの並び順は、画面上でドラッグして変更します。グループ内のリンク順は、そのグループの編集画面で並び替えます。

## 公開サイトへの表示

テンプレートに `<!-- BEGIN link --> ... <!-- END link -->` ブロックを置き、その中で `{link_list}` タグを使うと、登録したリンクが `<ul>` の HTML フラグメントとして出力されます。グループはネストされた `<ul>` として表現されます。

```html
<!-- BEGIN link -->
<aside class="blogroll">
  <h3>リンク集</h3>
  {link_list}
</aside>
<!-- END link -->
```

「非表示」にしたリンク・グループは `{link_list}` の出力から除外されます。

## SB2 / SB3 からの移行

SB2 の `link.cgi` や SB3 のリンクデータは、現在のインポータでは自動的に取り込まれません。インポート後、管理画面から手動で登録し直してください。
