# 公開設定と OG カード

「設定」では、ブログ名、説明、URL、管理画面の表示、AI 設定などを変更できます。権限によって表示されるタブは異なります。

## 画面設定

画面設定は、ログイン中のユーザーごとの管理画面設定です。

- 外観: 自動、ライト、ダーク
- 表示言語: 日本語、英語

ブログの公開ページではなく、管理画面の見え方に関する設定です。

## 基本設定

基本設定では、公開サイトに関わる情報を編集します。管理または上級ユーザーが利用できます。

| 項目 | 内容 |
|---|---|
| ブログタイトル | サイト名として表示されます |
| 説明 | サイト説明、フィード、OG description などに使われます |
| ベース URL | フィードや OG カードで絶対 URL を作るために使います |
| 言語 | 公開ページの言語やメッセージ表示に使います |
| llms.txt | AI エージェント向けの公開テキストを出すかどうか |

公開ブログでは、ベース URL を設定しておくことをおすすめします。未設定でも表示はできますが、フィードや SNS 共有時に URL が正しく解釈されないことがあります。

OG カードの既定背景・文字色は「デザイン設定」の「OG カード」タブで設定します。詳しくは下の [OG カード](#og-カード) を参照してください。

## llms.txt

llms.txt を有効にすると、公開記事の情報を AI エージェントが読みやすい Markdown として公開します。

- `/llms.txt`: 記事一覧のインデックス
- `/llms-full.txt`: 公開記事本文を含む全文

公開したくない場合は無効のままで構いません。無効時はこれらの URL は表示されません。

## OG カード

OG カードは、SNS などで記事 URL が共有されたときに使われる画像です。Serene Bach では、記事ごとに 1200 x 630 の画像を生成します。

ブログ共通の既定値 (背景画像と文字色) は、左メニューの「デザイン設定」>「OG カード」タブで編集します。記事ごとの上書きは記事編集画面で指定できます。

背景画像は次の順で使われます。

1. 記事ごとの OG カード背景 (記事編集画面で指定)
2. ブログ共通の OG カード背景 (デザイン設定 > OG カード)
3. 同梱の標準背景

背景画像は画像ライブラリから選びます。比率が合わない画像は中央を基準に切り抜かれます。

文字色は記事タイトルとブログタイトルの両方に適用されます。未設定の場合は標準の 2 色が使われます。背景画像の上に既に文字が入っているなど、OG カード側で文字を出したくない場合は「文字を表示しない」を選んでください。

## AI 設定

AI 設定では、AI 執筆補助と MCP アクセストークンを管理します。

AI 執筆補助を使うには、サーバ側に `SB_AI_SECRET` が設定されている必要があります。各ユーザーは、自分の API キーや利用するプロバイダを登録できます。

管理ユーザーは、MCP アクセストークンの発行と失効もこの画面で行えます。

## デザイン設定

テンプレート画面の「設定」では、公開サイトの表示方法を調整できます。

- アーカイブ、カテゴリー、タグページで使うテンプレート
- プロフィールページで使うテンプレート
- 一覧に表示する記事数
- 記事の並び順
- コメントの並び順
- 日付の表示形式

日付の表示形式は SB3 と同じ `%Year%`, `%Mon%`, `%Day%`, `%Week%`, `%Hour%`, `%Min%` などの書き方に対応しています。例: `%Year%-%Mon%-%Day% (%Week%)`
