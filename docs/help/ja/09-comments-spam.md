# コメントとスパム対策

Serene Bach では、記事ごとに読者からのコメントを受け付けられます。コメントは管理画面で確認し、必要に応じて承認、非公開、削除できます。

## コメント受付

「コメント」>「設定」で、ブログ全体の受付方法を選びます。

| 設定 | 内容 |
|---|---|
| 受け付ける | スパムチェックを通ったコメントをすぐ公開します |
| 承認制 | 管理者が承認するまで公開しません |
| 受け付けない | 新しいコメントを受け付けません |

初期設定は承認制です。公開直後のブログや、スパムが心配な場合は承認制のまま運用するのがおすすめです。

### 記事ごとのコメント受付け

ブログ全体の設定が「受け付ける」または「承認制」のときは、記事ごとに「コメント受付け」を個別にオフにできます。チェックボックスは記事編集画面の「ピン留め」の右側に表示され、初期値はオンです。

- オフにした記事では、公開ページの投稿フォームと既存コメントが非表示になり、新規投稿は拒否されます。
- ブログ全体が「受け付けない」のときは、記事側のチェックボックスは表示されません。全体設定が優先され、すべての記事のコメントが非表示になります。

## コメントの管理

「コメント」画面では、状態ごとにコメントを確認できます。

- 未承認
- 承認済み
- 非公開
- すべて

コメント本文を開くと、投稿者名、メールアドレス、URL、IP アドレス、本文を確認できます。

## スパム対策

Serene Bach には、追加設定なしで動く基本的なスパム対策が入っています。

- 人間には見えない入力欄を使った自動投稿の検出
- 速すぎる送信の拒否
- 同じ IP からの短時間の連続投稿制限
- URL や必須項目の検証

さらに、管理画面から次の設定ができます。

## IP ブラックリスト

拒否したい IP アドレスや範囲を、1 行に 1 つずつ入力します。

```text
198.51.100.5
198.51.100.0/24
2001:db8::1
```

一致した投稿は保存されません。

## スパム単語

投稿者名、メール、URL、本文に含まれてほしくない言葉を登録できます。1 行に 1 語ずつ入力してください。

```text
casino
viagra
```

一致した投稿は保存されません。

## Cloudflare Turnstile

Cloudflare Turnstile を使う場合は、サーバ側に次の環境変数を設定します。

```bash
SB_TURNSTILE_SITEKEY=0x4AAAAA...
SB_TURNSTILE_SECRET=0x4AAAAA...
```

両方が設定されている場合だけ、コメントフォームに Turnstile の確認が表示されます。

## いいねとスタンプ

コメントとは別に、記事には「いいね」と「スタンプ」を置けます。スタンプは `heart`、`laugh`、`wow`、`party` の 4 種類です。

テンプレートに対応するタグが置かれていれば、公開ページにボタンを表示できます。集計はアクセス解析画面で確認できます。
