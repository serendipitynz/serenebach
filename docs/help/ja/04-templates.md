# テンプレート編集

公開サイトの見た目は、テンプレートで変更できます。Go 版 Serene Bach でも、SB3 と同じ考え方のテンプレートを使います。既存の SB3 テンプレートは多くの場合そのまま取り込めますが、一部の古い機能には対応していません。

## テンプレートで編集できるもの

テンプレート編集画面では、主に次の内容を編集します。

- ページ全体の HTML
- 個別記事ページ用の HTML
- スタイルシート
- ロゴや背景画像などのテンプレート用ファイル

保存前にプレビューできるため、公開中のデザインをすぐ切り替えずに見た目を確認できます。

## 基本の書き方

テンプレートには、値を差し込むためのタグを書きます。

```html
<h1>{blog_title}</h1>
<article>
  <h2>{entry_title}</h2>
  {entry_description}
</article>
```

`{entry_title}` のような部分が、実際の記事タイトルや本文に置き換わります。

記事一覧、カテゴリー一覧、最近のコメントなど、複数件を表示する場所ではブロックを使います。

```html
<!-- BEGIN entry -->
<article>
  <h2>{entry_title}</h2>
  {entry_description}
</article>
<!-- END entry -->
```

`BEGIN` と `END` は独立した行に置いてください。

## よく使うタグ

| タグ | 内容 |
|---|---|
| `{blog_title}` | ブログタイトル |
| `{blog_description}` | ブログの説明 |
| `{blog_url}` | ブログの URL |
| `{site_css}` | テンプレートの CSS URL |
| `{site_parts}` | テンプレート用ファイルの URL |
| `{entry_title}` | 記事タイトル |
| `{entry_description}` | 記事本文 |
| `{entry_sequel}` | 追記 |
| `{entry_posted}` | 投稿日時 |
| `{entry_permalink}` | 記事の URL |
| `{entry_category_name}` | カテゴリー名 |
| `{entry_comments_count}` | コメント数 |
| `{entry_likes_count}` | いいね数 |
| `{entry_stamps_count}` | スタンプ数 |

SB3 互換の別名タグにも対応しているものがあります。古いテンプレートを取り込んだ後は、テンプレート編集画面のチェック結果を確認してください。

## テンプレート用ファイル

ロゴ、背景画像、追加 CSS などは、テンプレートのアセットとしてアップロードできます。アップロードしたファイルは `{site_parts}` を使って参照します。

```html
<img src="{site_parts}logo.png" alt="">
```

## デザイン設定

「デザイン設定」では、公開サイト全体に関わる表示設定を変更できます。

- 利用中テンプレートの切り替え
- アーカイブやカテゴリー一覧で使うテンプレート
- プロフィールページで使うテンプレート
- 一覧に表示する記事数
- 記事やコメントの並び順
- 日付の表示形式

## インポートとエクスポート

SB3 形式の `template.txt` をインポートできます。古いテンプレートで使われている Shift_JIS、EUC-JP、ISO-2022-JP などの文字コードは、取り込み時に UTF-8 へ変換されます。

エクスポートも `template.txt` 形式で行います。バックアップや別環境への移動に利用してください。

## 対応していない古い機能

トラックバック、Amazon アフィリエイト、携帯向け表示など、Go 版 Serene Bach では対応していない SB3 のタグがあります。取り込み自体はできますが、該当箇所はテンプレート編集画面で修正してください。

詳しくは [SB2 / SB3 からの移行と機能差異](sb3-migration) を参照してください。
