# Serene Bach ヘルプ

Serene Bach は、セルフホストで運用できるウェブログ管理ツールです。2000 年代の Serene Bach / SB3 が持っていた「自分のサーバに置いて、自分で管理する」考え方を引き継ぎながら、Go 製の単一バイナリとして再実装されています。

このヘルプでは、設置後によく使う管理画面の操作を中心に説明します。細かな内部仕様よりも、記事を書く、見た目を整える、コメントを管理する、既存の SB3 環境から移す、といった実際の作業に沿って読めるようにしています。

## このヘルプについて

管理画面の左メニューにある「ヘルプ」から、各ページを切り替えられます。ヘルプは Serene Bach 本体に同梱されているため、インターネットに接続していない環境でも読むことができます。

SB3 と Go 版 Serene Bach では、管理画面の雰囲気やテンプレートの考え方は似ていますが、コメント、トラックバック、画像、AI 連携などには違いがあります。移行前に [SB2 / SB3 からの移行と機能差異](sb3-migration) を確認してください。
