# カテゴリーとタグ

Serene Bach には、記事を整理するために「カテゴリー」と「タグ」があります。カテゴリーは記事を置く場所、タグは後から探すための目印として使うと整理しやすくなります。

## カテゴリーとタグの違い

| 項目 | カテゴリー | タグ |
|---|---|---|
| 記事ごとの数 | 1 つ | 複数 |
| 主な用途 | 日記、技術メモ、告知などの大分類 | Go、旅行、読書などの横断的な目印 |
| 作成方法 | カテゴリー画面で作成 | 記事保存時にも自動作成 |
| 階層 | 親カテゴリーを指定できます | 階層はありません |
| 公開ページ | カテゴリー別の記事一覧 | タグ別の記事一覧 |

## カテゴリーの管理

「カテゴリー」画面では、カテゴリーの追加、編集、削除、並び替えができます。

カテゴリーには説明文を設定できます。テンプレートが対応していれば、カテゴリーページの冒頭などに表示されます。

カテゴリーごとにテンプレートを指定することもできます。特定のカテゴリーだけ見た目を変えたい場合に利用します。

## タグの管理

タグは記事編集画面のタグ欄に入力すると自動的に作成されます。複数のタグを付ける場合は、カンマで区切って入力します。

「タグ」画面では、作成済みタグの名前や URL 用の名前を変更できます。タグを削除すると、そのタグと記事の関連付けも削除されます。記事そのものは削除されません。

## 使い分けの例

- 毎回 1 つだけ選びたい分類はカテゴリーにする
- 後から横断検索したい言葉はタグにする
- URL やテンプレートを分けたいものはカテゴリーにする
- 季節、イベント名、技術名など増えやすいものはタグにする
