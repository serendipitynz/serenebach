# SB3 からの移行と機能差異

Perl 版 Serene Bach 3 の SQLite データベースから、Go 版 Serene Bach へデータを取り込めます。すべての機能が 1 対 1 で移植されているわけではないため、移行後は記事、カテゴリー、テンプレートを確認してください。

## 移行前に用意するもの

- SB3 の SQLite データベース
- Go 版 Serene Bach のバイナリ
- 取り込み先の新しいデータベース
- SB3 で使っていた画像ファイルやテンプレートファイル

本番環境のデータを直接使わず、必ずコピーを作ってから試してください。

## 基本の手順

まず、取り込み先に管理ユーザーと標準テンプレートを作成します。サンプル記事を入れないようにしておくと、移行後の確認がしやすくなります。

```bash
SB_SEED_NO_SAMPLES=1 ./serenebach seed
```

次に、SB3 のデータベースを指定して取り込みます。

```bash
./serenebach import /path/to/sb3.db
```

取り込みが終わると、件数と警告が表示されます。警告はテンプレート内の未対応タグなどを確認するためのものです。

## 取り込まれるもの

| データ | 内容 |
|---|---|
| ブログ設定 | タイトル、説明、URL、言語など |
| カテゴリー | 親子関係を含めて取り込みます |
| 公開済み記事 | 本文、追記、日時、カテゴリー、キーワードなど |
| タグ | SB3 のキーワードから作成します |
| テンプレート | HTML、CSS、個別記事用 HTML を取り込みます |

取り込んだテンプレートは、すぐには利用中になりません。テンプレート画面で確認してから切り替えてください。

## 取り込まれないもの

| データ | 理由・対応 |
|---|---|
| ユーザー | パスワード形式が異なるため、Go 版で作り直してください |
| 下書き・非公開記事 | 標準の取り込みでは公開済み記事だけを対象にします |
| コメント | 現時点では移行対象外です |
| トラックバック | Go 版では対応しません |
| 画像 | ファイルを手動で移し、必要に応じて画像ライブラリへ登録してください |
| プラグイン | Perl 版プラグインは動作しません |
| Amazon 関連機能 | Go 版では対応しません |

## テンプレートの確認

Go 版 Serene Bach は SB3 テンプレートとの互換性を重視していますが、すべてのタグに対応しているわけではありません。

対応している代表例:

- 記事、カテゴリー、アーカイブの基本表示
- プロフィールページ
- コメントフォーム
- 最近の記事、最近のコメント、カテゴリー一覧、月別アーカイブ
- SB3 形式の日付フォーマット

対応していない代表例:

- トラックバック関連
- Amazon アフィリエイト関連
- 携帯向け表示関連
- 一部のおすすめ記事・選択記事系タグ
- SB3 プラグインに依存する表示

テンプレートを取り込むと、未対応または挙動が異なるタグについて警告が表示されます。公開前にテンプレート編集画面で確認してください。

## URL 互換

SB3 で使っていた一部の URL は、Go 版の URL へ転送されます。古い記事リンクやカテゴリーリンクをなるべく維持するための機能です。

ただし、すべての外部リンクや手書き URL を完全に再現できるわけではありません。重要なページは移行後に実際の URL を開いて確認してください。

## 画像の移行

SB3 の画像ファイルは自動では取り込まれません。まず、公開サイトから参照できる場所に画像をコピーしてください。

新しい記事で画像ライブラリを使いたい場合は、「画像」画面から改めてアップロードします。過去記事内に直接書かれた画像パスは、そのパスで画像が配信されていれば表示されます。

## SB3 と Go 版の主な違い

| 項目 | SB3 | Go 版 Serene Bach |
|---|---|---|
| 実行環境 | Perl / CGI | Go 製単一バイナリ、常駐サーバまたは CGI |
| データベース | SQLite | SQLite |
| テンプレート | SB3 テンプレート | SB3 互換テンプレートを中心に対応 |
| 記事フォーマット | HTML、SB3 記法など | HTML、Markdown、sbtext |
| コメント | コメント、トラックバック | コメントのみ。トラックバックは非対応 |
| 画像管理 | アップロード管理 | 画像ライブラリ、サムネイル、OG カード背景 |
| 静的再構築 | 対応 | 対応。画像とテンプレートアセットも出力 |
| XML-RPC | 対応機能あり | RSD はありますが XML-RPC 本体は未実装 |
| AI 連携 | なし | MCP、llms.txt、AI 執筆補助に対応 |
| プラグイン | Perl プラグイン | Perl プラグインは非対応 |

## 移行後に確認すること

1. 記事数とカテゴリーを確認します。
2. 公開サイトでトップページ、記事ページ、カテゴリー、アーカイブを開きます。
3. テンプレートの警告を確認し、未対応タグを修正します。
4. 画像パスが切れていないか確認します。
5. コメント受付設定、ブログ URL、OG カード設定を見直します。
6. 必要なユーザーを作り直します。
