# 画像アップロード

「画像」画面では、記事や OG カードで使う画像を管理できます。アップロードした画像は、公開サイトから `/img/...` の URL で参照されます。

## アップロード

画像画面にファイルをドラッグするとアップロードできます。クリックしてファイルを選ぶこともできます。複数ファイルをまとめてアップロードできます。

記事編集画面から画像を挿入する場合は、「画像を挿入」を使います。本文エリアへ画像を直接ドロップすると、アップロードと本文への挿入を一度に行えます。

## 対応形式

標準で対応している画像形式は次の通りです。

- JPEG
- PNG
- GIF
- WebP

1 ファイルあたりの上限は `SB_UPLOAD_MAX_MB` で変更できます。初期値は 10 MB です。

## 一覧表示

画像画面では、グリッド表示とリスト表示を切り替えられます。ファイル名で検索することもできます。

リスト表示では、ファイル名、サイズ、画像寸法などを確認できます。記事編集画面や OG カード背景の選択画面でも、同じ画像ライブラリを使います。

## 画像 URL のコピー

画像の URL をコピーして、記事本文やテンプレートへ貼り付けられます。

```html
<img src="/img/2026/04/sample-ab12cd.jpg" alt="">
```

Markdown 記事では、画像挿入ボタンを使うと Markdown の画像記法で挿入されます。

## alt テキスト

画像には alt テキストを設定できます。AI 執筆補助を設定している場合、アップロード後に alt テキストの候補を生成できます。

alt テキストは、画像が表示されない場合やスクリーンリーダーで読む場合に使われます。意味のある画像には、短い説明を入れておくことをおすすめします。

## 削除

画像を削除すると、ファイルも削除されます。削除した画像を参照している記事やテンプレートでは、画像が表示されなくなります。

削除前に、その画像を使っている記事がないか確認してください。

## 静的再構築との関係

静的再構築を行うと、アップロード画像も出力先へコピーされます。静的配信でも記事画像や OG カード画像をそのまま配信できます。
