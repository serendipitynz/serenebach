# ライブラリ

「ライブラリ」画面では、記事や OG カードで使うファイルを管理できます。アップロードしたファイルは、公開サイトから `/img/...` の URL で参照されます。

## アップロード

ライブラリ画面にファイルをドラッグするとアップロードできます。クリックしてファイルを選ぶこともできます。複数ファイルをまとめてアップロードできます。

記事編集画面からファイルを挿入する場合は、「ファイルを挿入」を使います。本文エリアへファイルを直接ドロップすると、アップロードと本文への挿入を一度に行えます。

## 対応形式

標準で対応しているファイル形式は次の通りです。

| 種類 | 拡張子 |
|------|--------|
| 画像 | `.png`, `.jpg`, `.jpeg`, `.gif`, `.webp` |
| 音声 | `.mp3`, `.ogg`, `.m4a` |
| 文書 | `.pdf`, `.txt`, `.md` |
| 動画 | `.mp4`, `.webm` |

1 ファイルあたりの上限は `SB_UPLOAD_MAX_MB` で変更できます。初期値は 10 MB です。

## 一覧表示

ライブラリ画面では、グリッド表示とリスト表示を切り替えられます。ファイル名で検索することもできます。

リスト表示では、ファイル名、サイズ、画像寸法（画像の場合）などを確認できます。記事編集画面や OG カード背景の選択画面でも、同じライブラリを使います。OG カード背景の選択画面では、画像のみが表示されます。

## ファイル URL のコピー

ファイルの URL をコピーして、記事本文やテンプレートへ貼り付けられます。種類に応じて、適切な HTML タグや Markdown 記法で挿入されます。

- 画像 … `<img src="..." alt="">` / `![alt](...)`
- 音声 … `<audio controls src="..."></audio>` / リンク
- 動画 … `<video controls src="..."></video>` / リンク
- 文書 … `<a href="..." download>filename</a>` / リンク

## alt テキスト

画像には alt テキストを設定できます。AI 執筆補助を設定している場合、アップロード後に alt テキストの候補を生成できます。

alt テキストは、画像が表示されない場合やスクリーンリーダーで読む場合に使われます。意味のある画像には、短い説明を入れておくことをおすすめします。

## 削除

ファイルを削除すると、ファイルも削除されます。削除したファイルを参照している記事やテンプレートでは、ファイルが表示されなくなります。

削除前に、そのファイルを使っている記事がないか確認してください。

## 静的再構築との関係

静的再構築を行うと、アップロードファイルも出力先へコピーされます。静的配信でも記事画像や OG カード画像をそのまま配信できます。
