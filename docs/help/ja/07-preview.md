# プレビュー機能

プレビュー機能を使うと、公開前の記事や、まだ利用中にしていないテンプレートの見た目を確認できます。SB3 のダイナミックプレビューに近い機能です。

## 下書き記事のプレビュー

記事編集画面の「プレビュー」を開くと、下書きや非公開の記事を公開ページと同じレイアウトで確認できます。

公開サイトにはまだ表示されないため、公開前の確認や、長い記事の表示崩れチェックに使えます。

## テンプレートのプレビュー

テンプレート一覧やテンプレート編集画面から、特定のテンプレートで公開サイトを表示できます。

テンプレートを「利用中」に切り替える前に確認できるため、公開中のデザインを壊さずに調整できます。

## 記事とテンプレートを組み合わせて確認する

下書き記事を、まだ利用中ではないテンプレートで表示することもできます。移行した SB3 テンプレートを調整するときや、カテゴリー専用テンプレートを試すときに便利です。

## 誰が見られるか

プレビューはログイン中の管理画面ユーザーだけが利用できます。ログインしていない状態でプレビュー用 URL を開いても、下書き記事や未適用テンプレートは表示されません。

プレビュー用のページは検索エンジンに登録されないように扱われます。また、静的再構築の出力には含まれません。
